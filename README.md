## Environment Used
This application is written in Next.JS v 10.1.
Node version 14.16.0
npm version 6.14.11

The demo can be seen here :
https://pokemon-dfc80.web.app

### Setup

1. Install all dependencies
```sh 
$ npm install
```
2. runing a development server.
```sh 
$ npm run dev
```
3. build to production
```sh 
$ npm run build
```
